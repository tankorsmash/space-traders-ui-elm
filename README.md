# SpaceTraders UI

A UI for [SpaceTraders](https://spacetraders.io/), built with [Elm](https://elm-lang.org/)

## Current Features

- auth
- view your Agent
- view your Contracts
- view (parts of) your ships
  - move ships between being docked and being in orbit
- view systems, in 3D!
- view waypoints
